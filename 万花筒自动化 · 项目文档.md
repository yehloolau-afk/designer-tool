## 项目定位

面向小学语文内容的**抖音短视频自动化生成工具**。输入任意一篇小学语文课文或文章，工具自动完成「分镜拆解 → AI 生图 → AI 生视频」全流程，输出可直接导入 AE 的视频素材片段。

**分工：**
- 🤖 工具负责：分镜脚本 + 画面生成 + 视频片段输出
- 🎬 用户负责：AE 合成 + 配音 + 字幕文案

**目标格式：** 抖音竖版 9:16，总时长 30–60 秒，单片段 3–5 秒

---

## 工作流

```
用户输入课文
     ↓
① 本地 JS 解析分镜（无需 API）
  按视觉关键词打分，选出最具画面感的句子
  生成：中文场景描述 / 英文提示词 / 镜头运动建议
     ↓
② 分镜预览 & 编辑
  可调整、修改提示词、增减场景
     ↓
③ 批量 AI 生图（Seedream 5.0）
  每个场景 → 1 张竖版图（9:16）
     ↓
④ 批量 AI 生视频（Seedance 1.5 Pro）
  每张图 → 3–5 秒视频片段（镜头运动注入提示词）
     ↓
⑤ 下载素材 → AE 合成
```

## 本地运行方式

双击 `启动万花筒-自动化.command`，自动启动两个本地服务：

| 服务 | 端口 | 作用 |
|------|------|------|
| python3 -m http.server | 8765 | 提供 HTML 文件访问 |
| proxy.py | 8766 | 转发火山方舟 API 请求（绕过浏览器 CORS） |

打开地址：`http://localhost:8765/万花筒-自动化.html`

> 注意：必须通过 localhost 打开，直接双击 html 文件（file://）会导致 API 调用失败。

## 技术选型

| 模块 | 技术 | 备注 |
|------|------|------|
| 分镜解析 | 纯 JS 本地解析 | 关键词打分 + ZH→EN 映射，零 API 调用 |
| 图片生成 | 火山方舟 Seedream 5.0（`doubao-seedream-5-0-260128`） | 9:16 竖版 |
| 视频生成 | 火山方舟 Seedance 1.5 Pro（`doubao-seedance-1-5-pro-251215`） | 异步轮询 |
| 前端 | 单文件 HTML + Apple 风格 CSS + 原生 JS | 无框架 |
| 本地代理 | Python proxy.py（localhost:8766） | 解决浏览器 CORS 问题 |
| 线上代理 | Netlify Functions（volc-img / volc-vid） | 部署后替代本地代理 |
| Key 存储 | localStorage | 与万花筒共享 `imggen_cfg.key` |
